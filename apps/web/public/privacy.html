Политика в отношении обработки персональных данных

1.2. Политика определяет порядок обработки персональных данных и меры по обеспечению безопасности персональных данных Пользователей сервиса «Зачёт», расположенного по адресу https://zachet.tech/.

1.3. Использование сервиса «Зачёт» означает безоговорочное согласие Пользователя с настоящей Политикой и указанными в ней условиями обработки его персональных данных.

2. Состав персональных данных

2.1. Оператор обрабатывает следующие персональные данные Пользователя:

  • Адрес электронной почты (e-mail);
  • Идентификатор пользователя в мессенджере Telegram (в случае авторизации через Telegram);
  • Данные о совершенных платежах (сумма, дата, идентификатор транзакции);
  • Файлы и тексты, созданные Пользователем в процессе использования сервиса.

2.2. Оператор не собирает и не обрабатывает данные о банковских картах Пользователя. Оплата производится через защищенный платежный шлюз банка-эквайера.

3. Цели обработки персональных данных

3.1. Обработка персональных данных осуществляется в следующих целях:

  • Предоставление Пользователю доступа к функционалу сервиса «Зачёт»;
  • Идентификация Пользователя в рамках соглашений и договоров с Оператором;
  • Обработка платежей за использование подписки;
  • Связь с Пользователем, включая направление уведомлений и информации, касающихся использования сервиса;
  • Улучшение качества сервиса и разработка новых функций.

4. Порядок и условия обработки персональных данных

4.1. Обработка персональных данных осуществляется без ограничения срока, любым законным способом, в том числе в информационных системах персональных данных с использованием средств автоматизации или без использования таких средств.

4.2. Оператор обеспечивает сохранность персональных данных и принимает все возможные меры, исключающие доступ к персональным данным неуполномоченных лиц.

4.3. Персональные данные Пользователя никогда, ни при каких условиях не будут переданы третьим лицам, за исключением случаев, связанных с исполнением действующего законодательства.

5. Сроки хранения персональных данных

5.1. Оператор осуществляет хранение персональных данных Пользователя, включая созданные им в сервисе работы, в течение 2 (двух) лет с момента последнего использования сервиса Пользователем.

  • Отозвать согласие на обработку персональных данных, направив соответствующий запрос Оператору.
